# בונה אפליקציות עסקיות 🏗️

סקיל Claude Code שבונה לך **אפליקציה עסקית שלמה באוויר** — CRM, ניהול לקוחות, דשבורד מדדים — בשיחה מודרכת בעברית. בלי רקע טכני, בלי קוד. מהשיטה של **טל בשור**: האפליקציה היא סיסטם של העסק — היא פותרת צוואר בקבוק ומודדת מדדים.

## התקנה

```bash
git clone https://github.com/talbs1988-dotcom/business-app-builder.git
cp -R business-app-builder/business-app-builder ~/.claude/skills/
```

פותחים סשן חדש של Claude Code — וזהו.

## איך מתחילים

פשוט אומרים לקלוד:

- **"תבנה לי CRM לעסק"**
- **"אני רוצה דשבורד מדדים"**
- **"תבנה לי מערכת ניהול לקוחות"**
- ואם עצרתם באמצע — **"איפה עצרנו?"** והבנייה ממשיכה מאותה נקודה

## מה מקבלים

- 🛡️ **לא נתקעים.** בדיקה אוטומטית בין כל שלב, נקודות שמירה, ומסלול חלופי מוכן לכל תקלה. סשן נפל? ממשיכים מאיפה שעצרנו.
- 🔐 **נעול כברירת מחדל.** מסך התחברות תמיד; אתם קובעים במייל מי נכנס — אתם והצוות, ואף אחד אחר. לפני כל עלייה לאוויר קלוד מנסה "לפרוץ" בעצמו ומוודא שנחסם.
- 🎨 **נראה מקצועי.** צבעי מותג, פונט עברי אמיתי, עברית מיושרת בלי פספוסים, ₪ ותאריכים בעברית.
- 📱 **מרגיש ישראלי.** כפתור וואטסאפ על כל לקוח, מסך בית עם המדדים שלכם, נתוני דוגמה בעברית מהרגע הראשון.
- 🔁 **ממשיך ללוות.** "האפליקציה לא עובדת" או "תוסיף פיצ'ר" — גם שבוע אחרי, קלוד קורא את מצב הפרויקט וממשיך משם.
